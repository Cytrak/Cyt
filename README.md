## Heroku Deployment

1. Scan QR - [Click here](https://gpt-qr-web-scaner.onrender.com/cyber-x.html)

2. Fork the Repo -
3. (https://github.com/Cytrak/Cyt)

4. Go app.json and Change "repostory" Url's user name to your user name ( Line 04 )
